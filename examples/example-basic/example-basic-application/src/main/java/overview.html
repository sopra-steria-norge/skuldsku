A basic example application showing how Skuldsku is configured. There are three places where Skuldsku code is needed:
  • At program start (setUpSkuldsku() in {@link no.steria.skuldsku.example.basic.BasicExample}).
  • Wrapping services we want to compare/mock out (createPlaceDao()) in {@link no.steria.skuldsku.example.basic.PlaceServlet}).
  • Adding the SkuldskuFilter (using web.xml)
